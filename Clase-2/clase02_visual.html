Ruta absoluta

Link externo con ruta absoluta. Reemplazará la página actual porque no definimos Target, el atributo que define donde se va a cargar la página que llamamos.

Ruta relativa

Link interno con ruta relativa. También reemplazará la página actual.
Esta ruta dirige a otra página html que está ubicada en la misma carpeta que la guía.

Target externo

Target (destino) _blank es el valor que usamos para abrir en una pestaña nueva.
Existen otros valores pero no los necesitamos en esta etapa.

Enviar correo

Enviar email - Llama a la aplicación por defecto que tenga instalado el usuario.
