Ejercitación – Clase 04

En este archivo vas a practicar las tres formas de aplicar CSS y los selectores básicos.

1) CSS inline

Modificá este párrafo usando el atributo style.

Probá cambiar color, tamaño o peso de la tipografía.

2) CSS dentro del documento (<style>)

Aplicá un color a todos los párrafos.

Aplicá un color distinto a los títulos h2.

3) CSS externo

Activá el <link> del head y creá un archivo styles.css.

Probá definir estilos generales para el body.

4) Selectores básicos

Modificá este usando una clase.

Modificá este usando un ID.

Sección de prueba

Modificá este usando un selector descendiente.

5) Colores

Probá aplicar colores usando:

  • nombre (red, blue, yellow)
  • rgb()
  • #hex
  • hsl()
